## 🔌 Backend Integration

### Supported Endpoints
✅ GET /quizzes - Fetch all quizzes  
✅ POST /quizzes - Create new quiz  
✅ POST /games/create/{quizId} - Create game session  
✅ PUT /games/start/{gameId} - Start game  
✅ PUT /games/complete/{gameId} - Complete game  
✅ GET /games - Get all games  
✅ GET /games/{gameId} - Get game details  

### WebSocket Support
✅ /ws - WebSocket endpoint  
✅ /app/answer - Send player answer  
✅ /topic/game/{gameId} - Receive game updates
